What is layout? (title) Web page layout is the creation of the structure of a hypertext document based on HTML markup, usually using style sheets and client scripts, so that design elements look similar to the layout. The layout of web pages differs from printing in that it is necessary to take into account the difference in the display of elements in different browsers and the difference in the size of the working space of devices. The process is complex and has a creative basis; none of the methods is canonical or accepted as a basis. All approaches to layout have both advantages and disadvantages.
